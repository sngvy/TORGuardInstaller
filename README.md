# TORGuardInstaller

**TORGuardInstaller — это скрипт, который автоматически установит и настроит официальную базу адресов Onionoo для защиты VPS от использования сети TOR.**

# Использование

**`bash <(curl -Ls https://raw.githubusercontent.com/sngvy/TORGuardInstaller/refs/heads/main/TORGuardInstaller.sh)`**
